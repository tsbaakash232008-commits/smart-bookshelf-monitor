# Technoverse Smart Library System

## Overview
A smart library detection and capture system built with Python, OpenCV, and Flask.

## Structure
- capture/: Contains the module for capturing camera images.
- detection/: Contains the PC image detection model.
- integrated/: Full system combining ultrasonic triggers, capture, and detection.
- ui/: Sentient Integrated Web UI application with Flask.
- ssets/: Contains placeholder images and additional resources.

## Setup
1. Install dependencies:
   `ash
   pip install -r requirements.txt
   `
2. For specific modules, run python main.py in their respective directories.
3. For the UI, run python app.py inside the ui/ directory.

## Environment Variables
If needed, create a .env file using .env.example as a template.
